## Installation
1. make sure python is installed
2. open cmd
3. \> cd "project folder"
4. \> pip install pipreqs
5. \> pipreqs . --> generates ***requirements.txt*** for python scripts in the folder
6. \> pip install -r requirements.txt && mycode.py
